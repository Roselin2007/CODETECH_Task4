# Project Overview

### Movie Search Engine

The **Movie Search Engine** is a web-based application that allows users to search for movies and retrieve detailed information instantly. By integrating with the **OMDb API (Open Movie Database API)**, the application fetches real-time movie data such as title, release year, genre, IMDb rating, poster image, and plot summary.

The project is built using **HTML, CSS, and JavaScript**, providing a responsive and interactive user interface. Users simply enter a movie name, and the application dynamically displays movie details without reloading the page.

### Key Features

* Search movies by title
* Fetch real-time movie information using OMDb API
* Display movie poster, rating, genre, and plot
* Responsive and user-friendly interface
* Error handling for invalid movie names
* Dynamic content updates using JavaScript
